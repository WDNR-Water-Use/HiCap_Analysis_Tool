Bases: object

Class to facilitate depletion or drawdown calculations

Objects from this class will have the required information needed to call any of the analysis methods in the package. The intent is that users will not have to access analytical solutions directly, but can set up a WellResponse object. The generation of WellResonse objects is generally done through an AnalysisProject object.

Class to calculate a single response for a single pumping well.
